# MultiMNIST overlapping digits (CapsNet)

**Source:** Sabour, Frosst & Hinton (2017), "Dynamic routing between capsules", NIPS.
**Demonstrates:** Capsules separately identify and reconstruct heavily overlapping objects (5.0% error on the dataset).

Two distinct-class MNIST digits each shifted up to 4 pixels and overlaid in 36x36 with 80% bounding-box overlap; 60M training / 10M test.
