# XOR

**Source:** Rumelhart, Hinton & Williams (1986), "Learning internal representations by error propagation", PDP Vol. 1, Ch. 8 / Nature short version.
**Demonstrates:** Backprop overcomes the Minsky-Papert single-layer limitation. Reported: ~558 sweeps to converge; only 2 of hundreds of runs hit a local minimum.

Architectures: 2-1-2 with skip connection, or 2-2-1.
