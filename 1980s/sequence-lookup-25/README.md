# 25-sequence look-up

**Source:** Rumelhart, Hinton & Williams (1986).
**Demonstrates:** RNN generalization. 5 input letters, 30 hidden, 3 output digits; 20 train / 5 test sequences. Net generalizes to 4-5 of 5 held-out sequences. A variable-time variant uses 60 hidden units to learn time-shift invariance.
