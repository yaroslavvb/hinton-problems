# Synthetic-spectrogram riser/non-riser discrimination

**Source:** Plaut & Hinton (1987), "Learning sets of filters using back-propagation", Computer Speech and Language.
**Demonstrates:** Backprop near Bayes-optimal performance on a controlled synthetic spectrogram task. 24 hidden units; 97.8% accuracy vs Bayes 98.8%.

Input: 6 frequency x 9 time = 54-D, with Gaussian noise.
