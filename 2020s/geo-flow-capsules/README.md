# Geo / Geo+ flow capsules

**Source:** Sabour, Tagliasacchi, Yazdani, Hinton & Fleet (2021), "Unsupervised part representation by flow capsules", ICML.
**Demonstrates:** Flow-based part discovery without object-level supervision.

Geo / Geo+: synthetic 2D shapes (ellipses, polygons) moving on textured backgrounds with ground-truth optical flow.
