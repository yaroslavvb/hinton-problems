# 4-2-4 encoder

**Source:** Ackley, Hinton & Sejnowski (1985).
**Demonstrates:** Hidden units self-organize into the optimal 2-bit binary code from local co-occurrence statistics.

Two groups V1, V2 of 4 visible units each, separated by 2 hidden units. Each pattern has one V1 unit on and the matching V2 unit on. Reported: solved in 250/250 runs (median 110 cycles).
