# Associative retrieval string toy

**Source:** Ba, Hinton, Mnih, Leibo & Ionescu (2016), "Using fast weights to attend to the recent past", NIPS.
**Demonstrates:** Fast weights as a form of attention to the recent past; outperforms IRNN, LSTM, and Associative-LSTM on sequential associative-retrieval tasks.

Strings like `c9k8j3f1??c -> 9` — multiple key/value pairs followed by a query key.
