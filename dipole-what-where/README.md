# Dipole what / where (discontinuous)

**Source:** Zemel & Hinton (1995).
**Demonstrates:** First explicit "what / where" toy in Hinton's corpus. Network spontaneously clusters horizontal-bar codes vs vertical-bar codes in opposite corners of a 2D implicit space. Compared to a Kohonen net.

Horizontal or vertical bars at random positions; 100 hidden units, 2D implicit space.
