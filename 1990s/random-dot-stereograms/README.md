# Random-dot stereograms (Imax)

**Source:** Becker & Hinton (1992), "A self-organizing neural network that discovers surfaces in random-dot stereograms", Nature.
**Demonstrates:** Spatial coherence as an unsupervised learning signal — two modules each see one strip; trained by maximizing mutual information between modules' outputs assuming a common cause.

The foundational unsupervised stereo demonstration. Re-proposed as a GLOM target task (Hinton 2021).
