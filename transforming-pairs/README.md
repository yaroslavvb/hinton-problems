# Synthetic image-pair transformations

**Source:** Memisevic & Hinton (2007), "Unsupervised learning of image transformations", CVPR.
**Demonstrates:** Gated three-way (input x output x hidden) multiplicative interactions discover transformation features. Hidden units learn Reichardt-like motion detectors when applied to real video frame pairs.

Pairs (x, y) where y is x transformed by a known random transformation (shifts, rotations, random-dot patterns).
