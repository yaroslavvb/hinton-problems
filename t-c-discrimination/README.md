# T-C discrimination

**Source:** Rumelhart, Hinton & Williams (1986).
**Demonstrates:** Early "convolutional" weight-tying constraint; the network discovers compactness detectors, bar detectors, on-centre/off-surround detectors.

8 patterns: block T and block C in 4 rotations on a 2D retina with shared 3x3 receptive fields.
