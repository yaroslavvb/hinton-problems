# Constellations dataset

**Source:** Kosiorek, Sabour, Teh & Hinton (2019), "Stacked capsule autoencoders", NeurIPS.
**Demonstrates:** Cleanest possible test of routing-by-attention without pixels. Each example is a union of K=3 unknown affine-transformed copies of fixed point templates (sets of 11 points = 4+4+3).
