# AIR Multi-MNIST

**Source:** Eslami, Heess, Weber, Tassa, Szepesvari, Kavukcuoglu & Hinton (2016), "Attend, Infer, Repeat", NIPS.
**Demonstrates:** Learn count, per-digit "what" (VAE latent), and per-digit "where" (affine) on scenes with a variable number of objects.

50x50 canvas with 0, 1, or 2 randomly placed/scaled MNIST digits.
